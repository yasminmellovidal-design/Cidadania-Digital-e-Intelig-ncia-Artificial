# Cidadania-Digital-e-Intelig-ncia-Artificial# Portal Conexão Segura 🌐

Interface acadêmica avançada voltada à conscientização comunitária a respeito de mídias sintéticas manipuladas por Inteligência Artificial (Deepfakes) e mitigação de desinformação digital.

## 🚀 Arquitetura e Engenharia de Software
- **HTML5 Estrutural**: Escrita semântica limpa contendo as divisões obrigatórias da página para máxima indexabilidade e acessibilidade.
- **CSS3 Dinâmico**: Layouts geridos com caixas flexíveis (`Flexbox`), transições nativas e `Media Queries` estruturadas para adaptação fluida em telas mobile.
- **JavaScript Funcional (DOM)**: Escuta de eventos em tempo real, lógica aritmética para processamento de variáveis e renderização condicional de dados sem erros no console.

## 🤖 Uso Ético e Transparência de IA
Documentação de suporte operacional baseada nos prompts estruturados aplicados ao longo do desenvolvimento técnico:

| Componente Gerado | Prompt Fornecido ao Assistente Inteligente |
| :--- | :--- |
| Sincronia de Links do Menu | "Escreva uma lógica Javascript que compare o valor de window.scrollY com o offsetTop de seções específicas e adicione uma classe 'active' ao link âncora correspondente." |
| Barra de Progresso Superior | "Gere um script dinâmico acionado pelo evento scroll do navegador que calcule a porcentagem da leitura da página e aplique-a ao width de uma div de posição fixa." |
